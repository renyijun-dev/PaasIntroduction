中国移动一级PaaS平台

PaaS平台支撑了业务系统的分布式、微服务化，通过自动化、敏捷化的能力，保障了业务系统的高可用和弹性扩缩容。

平台简介

Introduce

中国移动一级PaaS平台选择以Docker和Kubernetes为基础来构建PaaS平台方案。
提供操作简便的一键式服务自动化部署、统一配置管理、 应用的弹性扩缩容、微服务管控、DevOps工具链、资源/服务/容器等多维度综合监控、安全 管控等功能，并在此基础上持续发展Serverless、AI、区块链等创新技术平台。

运营现状

Operation Status

一级PaaS平台已建设K8S集群57个，节点规模超过4.2千个，提供49种技术组件，为B、O、M三域共计近30个业务系统提供了PaaS平台资源管理与供给、容器应用运行环境、技术组件服务，运行业务实例数近3万个，日均CI/CD开发部署超过300次。

核心功能

Core Function

  • 多租户集中管理

    在多集群多租户模式下，为不同租户分配合适的资源，再由租户为其用户分配细化的资源分区，用于业务服务的部署。

  • 资源管理

    计算资源、存储资源、网络资源、镜像资源等基础资源的管理。

  • 服务管理

    包括对多租户安全集中管控、应用部署、应用配置统一管理、基础应用市场等功能。

  • 微服务管控

    为不同类型的微服务提供差异化的管理策略,进行服务设计、编排、配置以实现多种场景的敏捷交付、快速部署、高可用、弹性扩展。

  • 智能运维

    通过融合 IT 数据,对监控、自动化、服务管控进行支持,对业务运营产生持续洞察、持续改进、持续优化、持续提升、持续反馈。
